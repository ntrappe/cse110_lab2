Data Structures:

📌 BINARY SEARCH TREE

Useful for:
Sorting data and everything, including rebalancing, can be done in O(log n).

Notes:
Generally use unique values (don't want duplicates).

Time Complexity:
Operation Average Worst
Insert O(log n) O(n)
Remove O(log n) O(n)
Search O(log n) O(n)

📌 ARRAYS

Useful for:
Storing and accessing sequential data, temporarily storing, as an IO buffer (when reading from a file), lookup table, and hack for multiple return values.

Notes:
Can implement a dynamic array from static ones by continuously allocating double the space once the array is full.

Time Complexity:
Operation Average Worst
Access O(1) O(1)
Append --- O(1)
Insert --- O(n)
Remove --- O(n)
Search O(n) O(n)

📌 LINKED-LISTS

Useful for:
Useful in lists, queues, and stacks. Often used for circular (round robin) or adjacency lists.

Notes:
Singly-linked will save up on space and it's simpler to implement but it can make it more difficult to access previous nodes. Doubly-linked takes up double the space since we have 2x the pointers but easier to traverse backwards.

Time Complexity:
Operation Average Worst
Insert O(1) O(1)
Remove O(n) O(1)
Search O(n) O(n)

Progamming Languages

(1) C: Procedural-oriented language; no references (just pointers), no function overloading, + memory management.

Basic Example: #include <stdio.h>
int main()
  printf("Hello World!");
  return 0;
}

To Run: 
gcc -o hi Helloworld.c
hi

(2) C++: Object-oriented language; references, pointers, function overloading + memory management.

Basic Example: #include <stdio.h>
int main()
  cout << "Hello World! << endl;
  return 0;
}

To Run: 
g++ -o hi Helloworld.c
./hi

(3) Python: Object-oriented/scripting language; pointers, function overloading + automatic garbage collection.

Basic Example: def hello():
  print("Hello World!)
hello()

To Run: 
python Helloworld.py

(4) Java: Object-oriented language; references, method overloading + automatic garbage collection.

Basic Example: public class HelloWorld {
  public static void main(String[]args) {
    System.out.println("Hello World!");
  }
}

To Run: 
javac Helloworld.java
java Helloworld
